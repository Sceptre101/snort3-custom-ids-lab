### **2. The Custom Rules**
**Filename:** `local.rules`
**Location:** `snort3-custom-ids-lab/rules/local.rules`

```snort
# 1. Large ICMP (Ping of Death Variant)
alert icmp any any -> $HOME_NET any (msg:"ATTACK DETECTED: Unusually Large ICMP Packet"; dsize:>800; classtype:misc-activity; sid:1000010; rev:2;)

# 2. SQL Injection (UNION SELECT)
alert tcp any any -> $HOME_NET 80 (msg:"WEB ATTACK DETECTED: SQL Injection Attempt (UNION SELECT)"; content:"UNION+SELECT", nocase; classtype:web-application-attack; sid:1000011; rev:2;)

# 3. Directory Traversal (../)
alert tcp any any -> $HOME_NET 80 (msg:"WEB ATTACK DETECTED: Directory Traversal Attempt (../)"; content:"../"; classtype:web-application-attack; sid:1000012; rev:2;)

# 4. Cleartext Password Leak
alert tcp any any -> $HOME_NET 80 (msg:"CREDENTIAL LEAK: Cleartext Password over HTTP"; content:"POST"; content:"password=", nocase, distance 0; classtype:policy-violation; sid:1000013; rev:2;)

# 5. FTP Anonymous Login
alert tcp any any -> $HOME_NET 21 (msg:"RECON DETECTED: FTP Anonymous Login Attempt"; content:"USER anonymous", nocase; classtype:suspicious-login; sid:1000014; rev:2;)

# 6. SSH Brute Force Attempt
alert tcp any any -> $HOME_NET 22 (msg:"ATTACK DETECTED: Potential SSH Brute Force Attempt"; flags:S; detection_filter:track by_src, count 5, seconds 10; classtype:suspicious-login; sid:1000016; rev:1;)
